語法、變數

不同於Python，C#程式每行命令必須以 ; 結尾，同時程式區塊必須包含於兩個 { } 之間。C#變數名稱同Python，變數名稱可由英文字或底線符號(_)開始，同時變數名稱大小寫英文字為不同的變數。程式註解單行使用 //，多行使用 /* */ 將註解範圍框起來。因為C#為強型態程式語言，變數設定為 資料類型 變數名稱; 或 資料類型 變數名稱1,變數名稱2; 或 資料類型 變數名稱=設定值; 


//單行註解

運算子

運算子(operator)用於變數與值的運算，常用運算子包含計算、設定、比較、邏輯、隸屬等，接下來每個類別進行介紹。

  • 計算運算子 +、-、*、/、%、++、-- ，除了加、減、乘、除之外， % 代表求餘數， ++ 代表增加1簡寫，-- 代表減少1簡寫。
  • 設定運算子 =、+=、-=、*=、/=、%= ，用於變數設定表示，以 x+=5為例，他代表 x=x+5。
  • 比較運算子 ==、!=、>、>=、<、<= ，用於條件比較分別為 等於、不等於、大於、大於等於、小於、小於等於。
  • 邏輯運算子 &&、||、! ，用於串接多個條件比較分別為 且、或、否定。
