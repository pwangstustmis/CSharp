SFML- 模組簡介

SFML(Simple and Fast Multimedia Library)為一個跨平台簡單快速的多媒體程式庫，程式庫包含Window、Graphics、Audio、Network、System模組。

  • Window模組：提供遊戲視窗定義、事件處理、使用者輸入等。
  • Graphics模組：提供2D遊戲物件。
  • Audio模組：提供聲音相關處理。
  • Network模組：提供網路傳遞功能。
  • System模組：提供系統相關服務功能。
接下來將介紹遊戲模組常用功能。 請先建立C# console應用程式，開啟工具-NuGet套件管理員-套件管理器主控台後，將下列命令貼製主控台並執行套件安裝。請注意：請各應用程式專案都必須安裝套件。 


Install-Package SFML.Net -Version 2.5.0

SFML- Window模組

Windows模組將涵蓋VideoMode、RenderWindow、Event、Keyboard、Mouse類別。使用Window模組必須在程式開始使用using SFML.Window;匯入Window模組。

  • VideoMode：用於建立遊戲視窗模式，使用者可以透過預設VideoMode.DesktopMode設定視窗模式遊戲視窗，或代入視窗寬與高來建立VideoMode遊戲視窗物件。 
    
VideoMode mode=new VideoMode(800,600); //設定寬800點、高600點螢幕模式
  • RenderWindow：用於建立遊戲渲染視窗，透過代入視窗模式建立RenderWindow物件。除此之外RenderWindow建構式支援過載(overload)，除標題外，他也可以加入Styles物件設定，用於設定不同遊戲視窗樣式。 
    
RenderWindow window=new RenderWindow(new VideoMode(800,600),"測試視窗"); //設定寬800點、高600點標題為"測試視窗"的選染視窗
RenderWindow window=new RenderWindow(ViewMode.DesktopMode,"測試視窗",Styles.Fullscreen); //設定遊戲視窗為全螢幕模式
    渲染視窗物件常用方法包含isOpen()、Close()、setVerticalSyncEnabled()、Draw()、Clear()、Display()。
    • isOpen()：傳回布林值(true/false)標示視窗是否開啟。
    • Close()：執行關閉視窗的動作。
    • setVerticalSyncEnabled()：一般代入布林值true，目的是避免殘影。
    • Draw()：會代入要繪製物件，目的是將物件加入畫面。
    • Clear()：清除視窗畫面內容。
    • Display()：顯示現在畫面內容。

事件

物件可藉由產生訊號來執行某些動作，這些訊號稱為事件(Event)，以上述程式為例，Window模組中的Keyboard物件中IsKeyPressed()可用於偵測使用者是否按下鍵盤，代入Keyboard.Key.Escape鍵值用於判斷使用者是否按下ESC鍵。除了Keyboard物件，Window模組也包含Mouse物件。

在C#中，除了向上述程式將事件偵測放在遊戲視窗迴圈中，我們也可以將事件所需執行程式附加於事件上，將上述遊戲迴圈中如下程式片段移除。並在遊戲迴圈前加入下列程式片段，一樣可以按下ESC鍵關閉遊戲視窗。下列程式將事件處理動作附加於事件本身。 


window.KeyReleased+= (sender, args) =>{
  if (args.Code == Keyboard.Key.Escape)
    window.Close();
};

練習：請修改上述程式並使用事件附加處理程式方式，當點擊滑鼠左按鈕關閉遊戲視窗。

在C#中，事件處理除直接撰寫程式或附加處理程式外，也可套用函數呼叫方式進行呼叫，以SFML視窗KeyPressed事件為例，下列程式片段將處理程式KeyProcess附加於SFML視窗KeyPressed事件。請注意：事件處理程式會代入兩個值object(物件)、SFML.Window.KeyEventArgs(事件類別)，物件sender為產生事件的對象(在本單元為渲染視窗)，由於代入值可能為空值(null)，因此在類別定義時須加上?，處理程式KeyProcess在傳送物件不為空值時，將傳送物件sender轉換為RenderWindow物件，之後判斷按下鍵值是否為ESC鍵，如果是，關閉渲染視窗。 


private static void keyProcess(object? sender,SFML.Window.KeyEventArgs evt)
{           
  if (sender!=null)
    {
      RenderWindow window = (RenderWindow)sender;
      if (evt.Code==SFML.Window.Keyboard.Key.Escape)
      window.Close();
    }
}
window.KeyPressed += new EventHandler(keyProcess);
完整程式如下： 

using System;
using SFML.Graphics; //匯入SFML模組
using SFML.Window;

namespace classes
{
  class Program
  {
    private static void keyProcess(object? sender,SFML.Window.KeyEventArgs evt)
    {            
      if (sender!=null)
      {
        RenderWindow window = (RenderWindow)sender;
        if (evt.Code==SFML.Window.Keyboard.Key.Escape)
          window.Close();
      }
    }
    static void Main(string[] args)
    {
      RenderWindow window = new RenderWindow(new VideoMode(980,800),"My Window");
      window.SetVerticalSyncEnabled(true); //避免殘影
      window.KeyPressed += new EventHandler<KeyEventArgs>(keyProcess);

      while (window.IsOpen) //主要遊戲迴圈
      {

        window.DispatchEvents(); //送出所有等待事件
        window.Clear(); //清除前一個畫面內容
        window.Display(); //顯示現在畫面內容
      }
    }
  }
}
滑鼠除點選或移動事件處理，遊戲經常需要取得滑鼠現在的位置，使用者可以利用Mouse.GetPosition()傳回滑鼠現在的位置，Mouse.GetPosition()可以代入遊戲視窗嫆於取得滑鼠在該遊戲視窗位置，位置值資料類型為Vector2i，它包含X、Y整數屬性。

SFML- Graphics模組

Graphics模組含許多類別，本單元將介紹常用類別包含CircleShape(圓形)、Color(顏色)、RectangleShape(方形)、Texture(材質)、Sprite(精靈-包含於場景中的二維圖像或動畫)、Text(文字)、Font(字型)。

SFML- Graphics模組- CircleShape(圓形)

顧名思義是用於建立圓形物件，方法或屬性包含：

  • CircleShape(半徑)：方法用於建立一半徑值的圓形。 
    • 
CircleShape ball=new CircleShape(20);
  • FillColor(填滿顏色)：屬性用於設定或讀取填滿顏色，顏色為Color物件。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
  • OutlineColor(邊框顏色)：屬性用於設定或讀取邊框顏色，顏色為Color物件。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
ball.OutlineColor=Color.Blue;
  • Origin(中心點)：屬性用於設定或讀取圓形中心點，預設為左上角(0,0)。座標點為System模組Vector2f(x,y)物件，x,y為圓形物件座標值。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
ball.OutlineColor=Color.Blue;
ball.OutlineThickness=2;
ball.Origin=new Vector2f(10,10); //將圓形中心由左上角移至圓形中心。必須匯入SFML System模組
  • Rotation(旋轉)：屬性用於設定或讀取旋轉角度。
  • Position(位置)：屬性用於設定或讀取圓形位置，位置為System模組Vector2f(x,y)物件，x,y為圓形物件座標值。
  • Scale(比例)：屬性用於設定或讀取圓形比例，比例為System模組Vector2f(x,y)物件，用於代表圓形在x,y的比例。
  • GetGlobalBounds()：屬性用於取得圓形物件在遊戲視窗範圍，因為要判斷遊戲物件是否接觸需要比較物件之間的範圍位置是否重疊，不須使用條件比較，SFML透過GetGlobalBounds()即可完成，下列程式片段使用GetGlobalBounds()的Intersect()方法，判斷兩個物件是否重疊並傳回布林值true、false。 
    
CircleShape ball1=new CircleShape(20);
ball1.FillColor=Color.Red;
ball1.Position=new Vector2f(100,100);
CircleShape ball12=new CircleShape(20);
ball2.FillColor=Color.White;
ball2.Position=new Vector2f(200,200);
if (ball2.GetGlobalBounds().Intersect(ball1.GetGlobalBounds()))
  ball1.FillColor=Color.Green;

以下程式以全螢幕方式在視窗中央顯示半徑為50、顏色為白色圓形圖。

            }
        }
    }
}

當使用者按下空白鍵，布林變數shot由false變成true，在遊戲迴圈中，當shot為true時，子彈位置每次減10，一直到子彈Y位置小於0時，將shot變為false。(沒有發射子彈時，子彈跟著碉堡走，發射子彈後，子彈X位置不變，Y位置每次減10，一直到Y值小於0為止) 


using SFML.System; //匯入SFML模組
using SFML.Graphics; 
using SFML.Window;

namespace classes
{
    class Program
    {
        static void Main(string[] args)
        {
            //使用全螢幕遊戲視窗
            RenderWindow window = new RenderWindow(VideoMode.DesktopMode, "遊戲標題",Styles.Fullscreen);
            window.SetVerticalSyncEnabled(true); //避免殘影
            window.SetFramerateLimit(60); //設定畫面更新率
            

            CircleShape _base=new CircleShape(50);
            uint _baseY = window.Size.Y - 25;

            _base.FillColor= Color.White;
            _base.Position=new Vector2f(window.Size.X/2-25,_baseY);

            CircleShape bullet = new CircleShape(10);
            bullet.FillColor = Color.Red;
            
            bool shot = false;


            while (window.IsOpen) //遊戲迴圈
            {
                //檢查使用者輸入
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //按下ESC鍵關閉遊戲
                    window.Close();
                if (Keyboard.IsKeyPressed(Keyboard.Key.Space)) //按下空白鍵發射子彈
                    shot = true;

                if (Mouse.IsButtonPressed(Mouse.Button.Left))
                    window.Close();

                window.DispatchEvents(); //啟動所有等待中事件

                //更新遊戲內容
                _base.Position=new Vector2f(Mouse.GetPosition().X,_baseY);
                if (!shot)
                    bullet.Position = new Vector2f(Mouse.GetPosition().X + 40, _baseY);
                if (shot)
                {
                    bullet.Position = new Vector2f(bullet.Position.X, bullet.Position.Y-10);
                    if (bullet.Position.Y < 0)
                        shot = false;
                }
                

                //清除前一畫面
                window.Clear(); //清除上一次畫面並設定背景顏色


                //繪製更新遊戲內容
                window.Draw(bullet);
                window.Draw(_base);

                //顯示現在畫面
                window.Display();
            }
        }
    }
}

發射多個子彈

當使用者要發射多個子彈，須將子彈以清單或陣列儲存，並動態改變其位置。藉由宣告List<CircleShape>並搭配清單元素新增及移除Add、RemoveAt。下列程式中變數shootertime值為10時，玩家才能發射子彈，目的避免整個螢幕都是子彈。 


using SFML.System; //匯入SFML模組
using SFML.Graphics; 
using SFML.Window;

namespace classes
{
    class Program
    {
        static void Main(string[] args)
        {
            //使用全螢幕遊戲視窗
            RenderWindow window = new RenderWindow(VideoMode.DesktopMode, "遊戲標題",Styles.Fullscreen);
            window.SetVerticalSyncEnabled(true); //避免殘影
            window.SetFramerateLimit(60); //設定畫面更新率
            

            CircleShape _base=new CircleShape(50);
            uint _baseY = window.Size.Y - 25;

            _base.FillColor= Color.White;
            _base.Position=new Vector2f(window.Size.X/2-25,_baseY);

            CircleShape bullet = new CircleShape(10);
            bullet.FillColor = Color.Red;
            List<CircleShape> bullets=new List<CircleShape>();

            int shoottime = 0; //用於設定子彈重置時間


            while (window.IsOpen) //遊戲迴圈
            {
                //檢查使用者輸入
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //按下ESC鍵關閉遊戲
                    window.Close();
                if (Keyboard.IsKeyPressed(Keyboard.Key.Space)) //按下空白鍵發射子彈
                {
                    if (shoottime>=10) //子彈重置當shoottime到10
                    {
                        bullet.Position= new Vector2f(Mouse.GetPosition().X + 40, _baseY);
                        bullets.Add(bullet);
                        shoottime = 0;
                    }
                }
                if (shoottime < 10)
                    shoottime++;

                if (Mouse.IsButtonPressed(Mouse.Button.Left))
                    window.Close();

                window.DispatchEvents(); //啟動所有等待中事件

                //更新遊戲內容
                _base.Position=new Vector2f(Mouse.GetPosition().X,_baseY);
                for (int i=0;i<bullets.Count;i++)
                {
                    bullets[i].Position=new Vector2f(bullet.Position.X, bullet.Position.Y - 10);
                    if (bullets[i].Position.Y<0)
                        bullets.RemoveAt(i);
                }
                
                //清除前一畫面
                window.Clear(); //清除上一次畫面並設定背景顏色

                //繪製更新遊戲內容
                for (int i = 0; i < bullets.Count; i++)
                {
                    window.Draw(bullets[i]);
                }
                   
                window.Draw(_base);
                
                //顯示現在畫面
                window.Display();
            }
        }
    }
}

SFML- Graphics模組- RectangleShape(方形)

顧名思義是用於建立方形物件，方法或屬性包含：

  • RectangleShape(new Vector2f(寬,高))：方法用於建立一設定寬、高值的方形。 
    • 
RectangleShape r=new RectangleShape(new Vector2f(30,20);
  • FillColor(填滿顏色)：屬性用於設定或讀取填滿顏色，顏色為Color物件。 
    • 
RectangleShape r=new RectangleShape(new Vector2f(30,20);
r.FillColor=Color.Red;
  • OutlineColor(邊框顏色)：屬性用於設定或讀取邊框顏色，顏色為Color物件。 
    • 
RectangleShape r=new RectangleShape(new Vector2f(30,20);
r.FillColor=Color.Red;
r.OutlineColor=Color.Blue;
  • Origin(中心點)：屬性用於設定或讀取圓形中心點，預設為左上角(0,0)。座標點為System模組Vector2f(x,y)物件，x,y為圓形物件座標值。 
    • 
RectangleShape r=new RectangleShape(new Vector2f(30,20);
r.FillColor=Color.Red;
r.OutlineColor=Color.Blue;
r.OutlineThickness=2;
r.Origin=new Vector2f(10,10); //將方形中心由左上角移至圓形中心。必須匯入SFML System模組

  • Scale(比例)：屬性用於設定或讀取方形比例，比例為System模組Vector2f(x,y)物件，用於代表方形在x,y的比例。
  • GetGlobalBounds()：屬性用於取得方形物件在遊戲視窗範圍，因為要判斷遊戲物件是否接觸需要比較物件之間的範圍位置是否重疊，不須使用條件比較，SFML透過GetGlobalBounds()即可完成，下列程式片段使用GetGlobalBounds()的Intersect()方法，判斷兩個物件是否重疊並傳回布林值true、false。 
    
RectangleShape r1=new RectangleShape(new Vector2f(20,40));
r1.FillColor=Color.Red;
r1.Position=new Vector2f(100,100);
RectangleShape r2=new RectangleShape(new Vector2f(20,40));
r2.FillColor=Color.White;
r2.Position=new Vector2f(200,200);
if (r2.GetGlobalBounds().Intersect(r1.GetGlobalBounds()))
  r1.FillColor=Color.Green;

SFML- Graphics模組- Font(字型)與Text(文字)

SFML利用Text物件來顯示文字，Text物件一定要設定外部字型，否則無法顯示。請至Google Fonts下載一ttf字型檔，之後透過Font物件載入字型，請注意：外部圖片檔位置必須使用絕對位置，載入後，設定Text物件Font、CharacterSize、Position、FillColor來設定文字字型、大小、位置及顏色。下面程式片段設定字型位置在D磁碟機中game檔案夾中Clocktest檔案夾中Tapestry-Regular.ttf，之後產生文字物件，並設定字型、大小25點、位置(10,10)、顏色為紅色。 


Font font = new Font("D:\\games\\Clocktest\\Clocktest\\Tapestry-Regular.ttf");

SFML- Graphics模組- Clock時間

以下列程式為例，當進行動畫時，動畫速度會取決於設備的好或壞，如果是高階機器，子彈會跑很快，如果是低階的機器，子彈就會跑的比較慢，為了讓不同的機器有相同的時間感受，程式將使用時間物件-Clock。 


using SFML.Graphics; //匯入SFML模組
using SFML.Window;
using SFML.System;

namespace classes
{
    class Program
    {

        {

            window.SetFramerateLimit(60); //設定畫面更新率

            RectangleShape bullet = new RectangleShape(new Vector2f(3, 20));
            bullet.FillColor = new Color(106,232,136);

            while (window.IsOpen) //遊戲迴圈
            {
                //檢查使用者輸入
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //按下ESC鍵關閉遊戲
                {
                    window.Close();
                }

時間物件利用時間物件.Restart()方法取得每個動畫間的時間差，時間差以clock.Restart().AsSeconds()(秒)、clock.Restart().AsMicroseconds()(百分之一秒)或clock.Restart().AsMilliseconds()(千分之一秒)為單位，取得動畫間的時間差，再由設計者預定的動畫速度來決定移動的距離。以下列程式為例，設定動畫速度speed每秒500點。 


aa
using SFML.Graphics; //匯入SFML模組
using SFML.Window;
using SFML.System;

namespace classes
{
    class Program
    {
        static void Main(string[] args)
        {
            //使用全螢幕遊戲視窗
            RenderWindow window = new RenderWindow(VideoMode.DesktopMode, "遊戲標題", Styles.Fullscreen);
            window.SetVerticalSyncEnabled(true); //避免殘影
            window.SetFramerateLimit(60); //設定畫面更新率

            RectangleShape bullet = new RectangleShape(new Vector2f(3, 20));
            bullet.FillColor = new Color(106,232,136);
            bullet.Position= new Vector2f(window.Size.X/2+5, window.Size.Y-30);

            const float speed = 500; //假設每秒跑500點
            Clock clock=new Clock(); //建立時間物件
            float deltatime; //設定時間差變數
            
            while (window.IsOpen) //遊戲迴圈
            {
                //檢查使用者輸入
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //按下ESC鍵關閉遊戲
                {
                    window.Close();
                }
                window.DispatchEvents(); //啟動所有等待中事件

                //更新遊戲內容
                deltatime = clock.Restart().AsSeconds(); //取得每個動畫的時間差並以秒計算

                if (bullet.Position.Y > 0)
                    bullet.Position =new Vector2f(window.Size.X / 2 + 5, bullet.Position.Y- deltatime * speed); //將Y座標依照每秒速度改變

                //清除前一畫面
                window.Clear(); //清除上一次畫面並設定背景顏色

                //繪製更新遊戲內容
                if (bullet.Position.Y>0)
                    window.Draw(bullet);

                //顯示現在畫面
                window.Display();
            }
        }
    }
}
