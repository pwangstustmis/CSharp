SFML- 模組簡介

SFML(Simple and Fast Multimedia Library)為一個跨平台簡單快速的多媒體程式庫，程式庫包含Window、Graphics、Audio、Network、System模組。

  • Window模組：提供遊戲視窗定義、事件處理、使用者輸入等。
  • Graphics模組：提供2D遊戲物件。
  • Audio模組：提供聲音相關處理。
  • Network模組：提供網路傳遞功能。
  • System模組：提供系統相關服務功能。
接下來將介紹遊戲模組常用功能。 請先建立C# console應用程式，開啟工具-NuGet套件管理員-套件管理器主控台後，將下列命令貼製主控台並執行套件安裝。請注意：請各應用程式專案都必須安裝套件。 


Install-Package SFML.Net -Version 2.5.0

SFML- Window模組

Windows模組將涵蓋VideoMode、RenderWindow、Event、Keyboard、Mouse類別。使用Window模組必須在程式開始使用using SFML.Window;匯入Window模組。

  • VideoMode：用於建立遊戲視窗模式，透過代入視窗寬與高來建立VideoMode物件。 
    
VideoMode mode=new VideoMode(800,600); //設定寬800點、高600點螢幕模式
  • RenderWindow：用於建立遊戲渲染視窗，透過代入視窗模式建立RenderWindow物件。除此之外RenderWindow建構式支援過載(overload)，除標題外，他也可以加入Styles物件設定，用於設定不同遊戲視窗樣式。 
    
RenderWindow window=new RenderWindow(new VideoMode(800,600),"測試視窗"); //設定寬800點、高600點標題為"測試視窗"的選染視窗
RenderWindow window=new RenderWindow(new VideoMode(800,600),"測試視窗",Styles.Fullscreen); //設定遊戲視窗為全螢幕模式
    渲染視窗物件常用方法包含isOpen()、Close()、setVerticalSyncEnabled()、Draw()、Clear()、Display()。
    • isOpen()：傳回布林值(true/false)標示視窗是否開啟。
    • Close()：執行關閉視窗的動作。
    • setVerticalSyncEnabled()：一般代入布林值true，目的是避免殘影。
    • Draw()：會代入要繪製物件，目的是將物件加入畫面。
    • Clear()：清除視窗畫面內容。
    • Display()：顯示現在畫面內容。
下列程式以全螢幕方式顯示960x480遊戲視窗，當使用者按下ESC鍵關閉時遊戲視窗。 

using SFML.Graphics;
using SFML.Window;

namespace classes
{
  class Program
  {
    static void Main(string[] args)
    {
      RenderWindow window = new RenderWindow(new VideoMode(960,480), "960x480視窗",Styles.Fullscreen);
      window.SetVerticalSyncEnabled(true); //避免殘影

      while (window.IsOpen) //主要遊戲迴圈
      {
        if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //檢查是否按下ESC鍵
        {
          window.Close();
        }

        window.DispatchEvents(); //啟動所有等待中事件
        window.Display(); //執行現在顯示結果
      }
    }
  }
}

事件

物件可藉由產生訊號來執行某些動作，這些訊號稱為事件(Event)，以上述程式為例，Window模組中的Keyboard物件中IsKeyPressed()可用於偵測使用者是否按下鍵盤，代入Keyboard.Key.Escape鍵值用於判斷使用者是否按下ESC鍵。除了Keyboard物件，Window模組也包含Mouse物件。

練習：請修改上述程式，將點擊鍵盤ESC鍵關閉遊戲視窗，改為點擊滑鼠左按鈕關閉遊戲視窗。

在C#中，除了向上述程式將事件偵測放在遊戲視窗迴圈中，我們也可以將事件所需執行程式附加於事件上，將上述遊戲迴圈中如下程式片段移除。 


if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //檢查是否按下ESC鍵
{
  window.Close();
}

並在遊戲迴圈前加入下列程式片段，一樣可以按下ESC鍵關閉遊戲視窗。下列程式將事件處理動作附加於事件本身。 


window.KeyReleased+= (sender, args) =>{
  if (args.Code == Keyboard.Key.Escape)
    window.Close();
};

練習：請修改上述程式並使用事件附加處理程式方式，將點擊鍵盤ESC鍵關閉遊戲視窗，改為點擊滑鼠左按鈕關閉遊戲視窗。

事件附加處理程式除直接撰寫程式外，也可套用函數呼叫方式， 以SFML視窗KeyPressed事件為例，下列程式片段將處理程式KeyProcess附加於SFML視窗KeyPressed事件。請注意：事件處理程式會代入兩個值object(物件)、SFML.Window.KeyEventArgs(事件類別)，物件sender為產生事件的對象(在本單元為渲染視窗)，由於代入值可能為空值(null)，因此在類別定義時須加上?，處理程式KeyProcess在傳送物件不為空值時，將傳送物件sender轉換為RenderWindow物件，之後判斷按下鍵值是否為ESC鍵，如果是，關閉渲染視窗。 


private static void keyProcess(object? sender,SFML.Window.KeyEventArgs evt)
{           
  if (sender!=null)
    {
      RenderWindow window = (RenderWindow)sender;
      if (evt.Code==SFML.Window.Keyboard.Key.Escape)
      window.Close();
    }
}
window.KeyPressed += new EventHandler(keyProcess);
完整程式如下： 

using System;
using SFML.Graphics; //匯入SFML模組
using SFML.Window;

namespace classes
{
  class Program
  {
    private static void keyProcess(object? sender,SFML.Window.KeyEventArgs evt)
    {            
      if (sender!=null)
      {
        RenderWindow window = (RenderWindow)sender;
        if (evt.Code==SFML.Window.Keyboard.Key.Escape)
          window.Close();
      }
    }
    static void Main(string[] args)
    {
      RenderWindow window = new RenderWindow(new VideoMode(980,800),"My Window");
      window.SetVerticalSyncEnabled(true); //避免殘影
      window.KeyPressed += new EventHandler<KeyEventArgs>(keyProcess);

      while (window.IsOpen) //主要遊戲迴圈
      {

        window.DispatchEvents(); //送出所有等待事件
        window.Clear(); //清除前一個畫面內容
        window.Display(); //顯示現在畫面內容
      }
    }
  }
}

SFML- Graphics模組

Graphics模組含許多類別，本單元將介紹常用類別包含CircleShape(圓形)、Color(顏色)、RectangleShape(方形)、Texture(材質)、Sprite(精靈-包含於場景中的二維圖像或動畫)、Text(文字)、Font(字型)。

SFML- Graphics模組- CircleShape(圓形)

顧名思義是用於建立圓形物件，方法或屬性包含：

  • CircleShape(半徑)：方法用於建立一半徑值的圓形。 
    • 
CircleShape ball=new CircleShape(20);
  • FillColor(填滿顏色)：屬性用於設定或讀取填滿顏色，顏色為Color物件。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
  • OutlineColor(邊框顏色)：屬性用於設定或讀取邊框顏色，顏色為Color物件。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
ball.OutlineColor=Color.Blue;
  • Origin(中心點)：屬性用於設定或讀取圓形中心點，預設為左上角(0,0)。座標點為System模組Vector2f(x,y)物件，x,y為圓形物件座標值。 
    • 
CircleShape ball=new CircleShape(20);
ball.FillColor=Color.Red;
ball.OutlineColor=Color.Blue;
ball.OutlineThickness=2;
ball.Origin=new Vector2f(10,10); //將圓形中心由左上角移至圓形中心。必須匯入SFML System模組
  • Rotation(旋轉)：屬性用於設定或讀取旋轉角度。
  • Position(位置)：屬性用於設定或讀取圓形位置，位置為System模組Vector2f(x,y)物件，x,y為圓形物件座標值。
  • Scale(比例)：屬性用於設定或讀取圓形比例，比例為System模組Vector2f(x,y)物件，用於代表圓形在x,y的比例。

以下程式在960x480視窗中央顯示半徑為20、顏色為紅色圓形圖。 


using SFML.Graphics; //匯入SFML模組
using SFML.Window;

namespace classes
{
    class Program
    {

        static void Main(string[] args)
        {
            RenderWindow window = new RenderWindow(new VideoMode(960,480), "960x480視窗",Styles.Fullscreen);
            window.SetVerticalSyncEnabled(true); //避免殘影
            CircleShape ball=new CircleShape(20);
            ball.FillColor = Color.Red;
            ball.Position = new SFML.System.Vector2f(678, 374);
            
            while (window.IsOpen) //主要遊戲迴圈
            {
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //檢查是否按下ESC鍵
                {
                    window.Close();
                }
                window.DispatchEvents(); //啟動所有等待中事件
                window.Clear();
                window.Draw(ball);
                window.Display(); //執行現在顯示結果
            }
        }
    }
}

Vector2f mousePos=window.MapPixelToCoords(Mouse.GetPosition());
