SFML- 套件安裝與測試

SMFL提供簡單套件功能system、window、graphics、audio、network用於製作遊戲與多媒體應用。我們將整合SFML套件於C#物件導向程式設計。

套件安裝

請先建立C# console應用程式，開啟工具-NuGet套件管理員-套件管理器主控台後，將下列命令貼製主控台並執行套件安裝。請注意：請各應用程式專案都必須安裝套件。 


Install-Package SFML.Net -Version 2.5.0

套件測試

請將下列程式複製並貼於專案中 Program.cs 後執行，程式會開啟 640x480 藍色視窗。 


using System;
using SFML.Graphics; //匯入SFML模組
using SFML.Window;

namespace classes
{
    class Program
    {
        const int WIDTH = 640;
        const int HEIGHT = 480;
        const string TITLE = "套件測試";
        
        static void Main(string[] args)
        {
            VideoMode mode = new VideoMode(WIDTH, HEIGHT); //宣告2D模式
            RenderWindow window = new RenderWindow(mode, TITLE); //建立對應模式視窗物件

            window.SetVerticalSyncEnabled(true); //避免殘影

            window.Closed += (sender, args) => window.Close(); //設定視窗關閉事件

            while (window.IsOpen) //主要遊戲迴圈
            {
                if (Keyboard.IsKeyPressed(Keyboard.Key.Escape)) //檢查是否按下ESC鍵
                {
                    window.Close();
                }

                window.DispatchEvents(); //啟動所有等待中事件
                window.Clear(Color.Blue); //清除上一次畫面並設定背景顏色
                window.Display(); //執行現在顯示結果
            }
        }
    }
}
