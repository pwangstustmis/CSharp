輸入輸出

因為C#為一物件導向語言(在單元物件導向介紹與實作介紹)，主控台輸入輸出會使用 Console 物件。

  • Console.Read() 從鍵盤讀入一個鍵並將建值以整數(int)資料類型值傳回。
  • Console.ReadLine() 從鍵盤讀入一串鍵值並以字串(string)資料類型值傳回。
  • Console.Write(輸出值)或Console.Write(輸出格式,輸出值) 將輸出值顯示於主控台並不跳行或使用指定輸出格式不跳行。輸出格式設定為 {輸出值位置,輸出值長度:輸出值類型}，輸出值位置代表後續地幾個輸出值，第一個位置值為0；輸出值長度為正負值，數值代表長度，正值靠右對齊，負值靠左對齊；輸出值類型 c數字 代表貨幣、數字代表小數位，d數字 代表整數、數字代表長度，f數字 代表具小數數字、數字代表小數位。
  • Console.WriteLLine(輸出值)或Console.WriteLine(輸出格式,輸出值) 將輸出值顯示於主控台並跳行或使用指定輸出格式跳行。輸出格式設定為 {輸出值位置,輸出值長度:輸出值類型}，輸出值位置代表後續地幾個輸出值，第一個位置值為0；輸出值長度為正負值，數值代表長度，正值靠右對齊，負值靠左對齊；輸出值類型 c數字 代表貨幣、數字代表小數位，d數字 代表整數、數字代表長度，f數字 代表具小數數字、數字代表小數位。 


//讀入一個鍵值到整數變數x
int x=Console.Read();
//輸出x值(鍵值整數值)
Console.Write(x);
//輸出x值(文字鍵)，利用(char)x將整數x轉換為文字值
Console.Write((char)x);
//輸出x值(文字鍵)，10自長度、靠右對齊
Console.Write("{0,10}",(char)x);
//輸出x值(文字鍵)，10自長度、靠左對齊
Console.Write("{0,-10}",(char)x);
//輸出x值(文字鍵)，10自長度、靠左對齊
Console.Write("{0:d6},{1:f3}",10,10.1);

資料類型操作

資料類型包含 int(整數)、long(長整數)、float(浮點數)、double(雙浮點數)、bool(布林值true, false)、char(字元，必須用單引號括起)、string(字串，必須用雙引號括起)、陣列與清單(List)。由於每個資料類型均為物件導向類別，因此每個資料類型均包含一些設定的操作方法。

  • int(整數)： 最常用方法為ToString()，他會將整數字轉為字串。
  • long(長整數)： 長整數同整數，只是長整數有8位元組大小，整數有4位元組大小。
  • float(浮點數)： 具小數，設定值時最後必須加上英文字 f，代表是浮點數。
  • double(雙浮點數)： 具小數有，有8位元組大小。
  • 陣列： 陣列宣告為 資料類別[] 變數名稱;。起始設定為 資料類別[] 變數名稱=new 資料類別[陣列長度]; 或 資料類別[] 變數名稱=new 資料類別[] {以逗點隔開陣列值};
  • 清單： 清單宣告為 List<資料類別> 變數名稱;。起始設定為 List<資料類別> 變數名稱=new List<資料類別>(); 或 List<資料類別> 變數名稱=new List<資料類別>() {{以大括號括起個別清單值}};
  • var(隱含類型)： 根據代入值來設定資料類型，可藉由變數GetType()函數取得變數資料類型。 

//宣告整數變數x
int x;
//宣告整數變數y並設定起始值5
int y=5;
//宣告浮點變數a並設定值為1.1
float a=1.1f;
//宣告雙浮點變數b並設定值為1.1
double b=1.1;
//宣告布林變數c
bool c;
//宣告字元變數d 並設定值為字元a
char d='a';
//宣告字串變數e 並設定值為字元STUST
string e="STUST";
//宣告3元素整數陣列變數f
int[] f=new int[3];
f[0]=1; f[1]=2; f[2]=3;
//宣告整數陣列變數g並設定起始值
int[] g=new int[] {1,2,3};
//宣告整數清單變數h
List<int> h=new List<int>();
//清單能透過 Add, Append, Insert將元素加入清單
h.Add(1); h.Add(2); h.Add(3);
//宣告整數清單變數i並設定起始值
List<int> i=new List<int>(){{1},{2},{3}};
//隱含類型m
var m="字串";
Console.WriteLine(m.GetType());
